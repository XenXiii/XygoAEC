# Phase 0 Security Gates

Phase 0 cannot be accepted unless:

- staged mode fails closed
- production targets are blocked
- live credential-like values are rejected
- repository governance docs exist
- architecture ADR exists
- source ledger uses official sources
- staged synthetic fixtures exist
- automated tests pass

## Initial Security Target

Target baseline:
- OWASP ASVS-oriented web security posture
- NIST AI RMF-aligned AI governance constraints

These are directional controls for Phase 0, not a claim of full certification or full control coverage yet.
